# UniClaro

Web app mobile herramientas universitarias para notas, indice academico, faltas, estudio y tareas

## Desarrollo

```bash
npm install
npm run dev
```

## Produccion

```bash
npm run build
```

El proyecto esta preparado para publicarse en Vercel. `vercel.json` redirige las URLs de la SPA a `index.html` para que rutas como `/calculadora-nota-final` funcionen al abrirse directamente.
